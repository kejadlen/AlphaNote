# Xcode

## Useful Shortcuts

### UI

`⌘0`: Show/hide left sidebar
`⌘⌥0`: Show/hide right sidebar
`⌘Y`: Show/hide debug area

`⌘⌥[`,`⌘⌥]`: Move selection up/down a line
`⌃I`: Re-indent selection
`⌘⌥⌃G`: Run last test
